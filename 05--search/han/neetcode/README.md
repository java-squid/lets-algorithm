## enumerate() 
```python
for entry in enumerate(['A', 'B', 'C']):
    print(entry)

#(0, 'A')
#(1, 'B')
#(2, 'C')

for i, letter in enumerate(['A', 'B', 'C']):
    print(i, letter)

# 0 A
# 1 B
# 2 C
```
