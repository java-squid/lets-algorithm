# lets-algorithm

## 기간
- 2025년 8월 20일 ~ 11월 20일

## 방법
- 특정 범위 설정하고, 시간 내 풀고 공유 (optional)
  - 범위는 나눠서 풀고, 
  - 괜찮은 문제들 공유하고, rough 하게 공유

## 사이클
- 2주 단위

## 구성
- `/ds` (챕터)
  - `/han` (참가자명)
     - `백준문제번호.py` or `백준문제번호.md` or
     - `/etc` 추가문제의 경우...
     
## 풀이방식
- 우선 자신 문제 풀기 
- 최대 30분 고민 후 문제 풀기 (문제 이해, 정리, 코드 작성)
- 이 이상 소요된다면 교재 혹은 답안 보고 품.
